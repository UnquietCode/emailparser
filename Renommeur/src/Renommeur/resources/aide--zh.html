通过这个工具，可以很方便的对照片文件进行重命名操作。它还可以帮助你将照片序列化 (此功能可以在菜单栏: ' 文件 ' 设置是否开启), 除此之外，它还可以帮助你轻松实现照片文件的排序、删除照片、查看照片文件信息等等。
使用此工具，第一步要先设置照片来源文件夹，然后根据您的需要执行类似上述的功能，最后如果您确定修改，执行保存操作即可， 工具将自动保存更改信息到文件。保存操作的快捷键是：(Ctrl + Alt + Enter)。

'重命名' 标签:
  • 在照片下方文本框内输入文件名. 如果多个文件中包含相同的文字，您可以选中该相同的文字，以此创建一个按钮，然后 鼠标右击 或 F2。
  • 您还可以使用鼠标的三个操作：
    • 鼠标左击 : 选中查看项 ;
    • 鼠标中键 :' 重命名 ' 标签 和 ' 照片管理 ' 标签 间的切换 ;
    • 鼠标右击 : 将照片标记为删除。
  • 可以对照片进行如下操作 :
    • 标记为删除状态 : 'Del' 键 或 鼠标右击。同样，您还可以还原此操作，将照片状态更改为未删除 : 通过菜单栏 文件) ；
    • 通过点击鼠标中键查看 ' 照片管理 ' 标签。
'照片管理' 标签 :
  • 您可以通过 拖曳 更改照片的排列顺序；
  • 通过 鼠标右击 更改照片文件的状态（已删除/未删除）；
  • 通过点击 鼠标中键 切换到' 重命名 '标签。 同时在此标签中选中的项会显示在' 重命名 '标签。
