With this software, you can easily rename a set of pictures without typing all the words. It can also numbers the pictures (if you want to do it : File menu), changes the order, deletes some images ans shows  some exif informations. In a first time, you only prepared the pictures and the real modifications on files will only be done at the end (Ctrl + Alt + Enter).

'Rename' tab:
  • You enter the name of the picture in the text field area. If one or more words are often used, create a button by selecting the words and then right button or F2.
  • You can do three operations on a button with the mouse :
    • left button : to write at the end, or at the position of the caret ;
    • central button : to write at the end, preceded with a coma ;
    • right button : to remove it.
  • On the picture, you can do the following actions :
    • Mark the picture to be deleted : 'suppr' key or right button. You can remove the mark by doing the same thing (if you asked the deleted marked picture to be shown : File menu) ;
    • Show the 'Manage pictures' tab with a central button click.
'Manage pictures' tab :
  • You can change the order of the pictures with drag and drop ;
  • Mark a picture to be deleted (or not) with a rigth button click ;
  • Come back to 'Rename' tab with a central button click . The picture shown is the one you selected in the 'manage picture' tab.
